The Cyber Crime Awareness & Reporting Portal is a web application designed to educate users about different types of cybercrimes and provide a secure platform for reporting incidents. The system also includes awareness quizzes, alerts about trending scams, and an AI chatbot for user assistance.

The main goal of this project is to increase public awareness of online threats and make it easier for victims to report cybercrimes to the appropriate authorities.
